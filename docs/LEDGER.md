# 기록 대장 (LEDGER) — 결함·이탈 전수

이 저장소의 모든 **결함 보정 기록(C)**과 **구현 이탈 기록(D)**을 한 곳에 모은 대장이다.
종전에는 `corpus_md/README.md`(결함)·`index/README.md`·`server/README.md`(이탈)에 분산돼
있었고, 본 대장으로 이관하며 전역 번호를 재부여했다. 각 항목의 전체 서술은 원커밋의
해당 README 판에서 복원할 수 있다.

- **C(결함)**: 원본·추출·변환 단계의 데이터 결함과 그 보정. 원문 훼손이 아니라 추출 오류의 복원이다.
- **D(이탈)**: 지시서·규약 대비 근거 있는 구현 이탈 또는 규칙의 구체화. "README 기록을 조건으로 허용"
  원칙의 기록부다.
- 각 항목: **일자 · 위치 · 원커밋 · 요지**.
